# House Price Prediction - Ames Housing Dataset

## Project Overview
This project aims to predict house prices using the Ames Housing dataset. It involves data extraction, cleaning, exploratory data analysis, and visualization using Tableau.

## Structure
- `data/`: Raw and processed datasets.
- `notebooks/`: Jupyter notebooks for different stages of the pipeline.
- `scripts/`: ETL pipeline scripts.
- `tableau/`: Tableau dashboard links and project reports.

## Timeline
- **April 20, 2026**: Data extraction and initial cleaning.
- **April 21, 2026**: EDA, feature engineering, and final report.
